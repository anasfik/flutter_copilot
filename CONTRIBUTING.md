# Contributing

Thanks for helping improve `flutter_copilot`.

## Before You Start

- Open a GitHub issue for bugs, feature ideas, or larger changes.
- Keep pull requests focused and small.
- Match the existing Dart and Flutter style.

## Local Checks

Run these before opening a pull request:

```bash
flutter analyze
flutter test
```

For example changes:

```bash
cd example
flutter analyze
```

## Pull Requests

Include:

- What changed.
- Why it changed.
- Any manual testing you did.

This project is intended to be developed on GitHub, so issues and pull requests are the preferred place for discussion.
